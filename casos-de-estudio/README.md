# Casos de estudio — Portfolio

Cada archivo de esta carpeta documenta un proyecto o decisión relevante para construir el portfolio.

## Estructura de cada caso

- **El problema** — qué situación había que resolver
- **El contexto** — producto, equipo, constraints
- **Las decisiones** — qué elegimos y por qué (con alternativas descartadas)
- **El resultado** — qué cambió, qué aprendimos
- **Materiales** — links a Figma, capturas, docs relacionados
